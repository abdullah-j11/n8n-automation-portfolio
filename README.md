1. Pizza Order Automation System

What it does:
Automatically captures customer orders from a web form and stores them instantly in Airtable — no manual data entry, no missed orders, no spreadsheet updating.

Problem it solves:
Small food businesses lose orders and waste 1-2 hours daily managing orders manually across WhatsApp messages, phone calls, and paper notes. This workflow centralises everything automatically.

How it works:
1. Customer fills out an order form (name, items, quantity, delivery address)
2. N8N receives the form submission instantly via webhook trigger
3. Order details are automatically saved to an Airtable database
4. Business owner sees every order organised in one place in real time

Tools used:
- N8N (workflow automation)
- Airtable (order database)
- Webhook trigger (form capture)

Business value:
- Zero manual data entry
- Orders captured 24/7 even when staff are unavailable
- No orders lost or forgotten
- Full order history searchable in Airtable
